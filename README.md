Assistente de Voz Inteligente com RAG (Gemini + Whisper)

Este projeto apresenta um assistente de voz avançado capaz de "ler" e compreender documentos PDF específicos para responder perguntas via áudio. Utilizando a arquitetura RAG (Retrieval-Augmented Generation), o assistente não depende apenas do conhecimento geral da IA, mas consulta uma base de dados local (seu documento) para fornecer respostas precisas e fundamentadas.

O projeto foi desenvolvido como uma evolução de um desafio de curso, integrando tecnologias de ponta em Processamento de Linguagem Natural (NLP) e Speech-to-Text.
